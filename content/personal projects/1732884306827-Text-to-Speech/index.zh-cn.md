这个项目是一个使用 ElevenLabs API 将文本转换为语音的网络应用。它作为 App Masters 项目的一部分开发，使用了现代技术，包括 React、TypeScript 和 Ant Design。

[演示](https://app-masters-elevenlabs-project.vercel.app/)

## 功能

- `文本输入`：允许用户输入要转换为语音的文本。
- `语音选择`：用户可以从可用的不同语音中进行选择。
- `语音筛选`：用户可以按类别、年龄、性别、口音和描述筛选语音。
- `音频播放`：用户可以在应用内直接收听文本转语音的结果。
- `音频下载`：用户可以以 MP3 格式下载生成的音频。

## 使用的技术

- `React`：用于构建用户界面的 JavaScript 库。
- `TypeScript`：为代码添加静态类型的 JavaScript 超集。
- `Ant Design`：React 的 UI 组件库。
- `ElevenLabs API`：用于文本转语音的服务。
- `Vercel Blob Storage`：用于实现下载功能的文件存储服务。
- `Vercel`：用于部署 Web 应用的平台。
