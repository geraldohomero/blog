This project provides a simple Python script that allows you to export your movie collection from [Filmow](https://filmow.com) to [Letterboxd](https://letterboxd.com). By using the `beautifulsoup4`, `requests`, `pandas` and `regex` libraries, the script scrapes your Filmow collection and formats it for easy import into Letterboxd.
