Este projeto oferece um script simples em Python que permite exportar sua coleção de filmes do [Filmow](https://filmow.com) para o [Letterboxd](https://letterboxd.com). Utilizando as bibliotecas `beautifulsoup4`, `requests`, `pandas` e `regex`, o script faz o scraping da sua coleção no Filmow e a formata para facilitar a importação no Letterboxd.
