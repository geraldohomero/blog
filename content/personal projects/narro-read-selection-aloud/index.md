Narro is a text-to-speech reader for GNOME/Wayland using edge-tts. Select text in any application, copy it with Ctrl+C, and use a keyboard shortcut to open the TTS player — with Play/Pause/Stop controls, language/voice selector, and speed control.

### Key Features
* **Play / Pause / Stop:** Full reproduction controls.
* **20+ Neural Voices:** Portuguese (BR/PT), English, Spanish, French, German, and more.
* **Speed Control:** Speed adjustments from -50% to +100%.
* **Native GTK Interface:** Seamless visual integration with GNOME (including dark theme).
* **Editable Text:** Edit the captured text before playing it.
* **Status Bar:** Real-time visual feedback.
* **Toggle:** Press the shortcut again to quickly close the player.
